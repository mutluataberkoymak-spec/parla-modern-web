Sahibinden Entegrasyonu

JSON dosyasından portföy aktar

Sahibinden’den dışa aktardığımız JSON dosyasını buraya yükle. Sistem ilan başlığı, fiyat, konum, özellikler, açıklama ve fotoğrafları Parla portföy formatına çevirir.

  • Desteklenen biçim: JSON array veya {"listings": [...]}
  • Tekrarlı ilanlar ilan_no / id ile güncellenir.
  • Fotoğraf URL’leri varsa kart görseli olarak kullanılır.
  • Bu demo sürümde kayıtlar tarayıcı hafızasına eklenir.
